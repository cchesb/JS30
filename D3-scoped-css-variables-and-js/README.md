# Scoped CSS Variables and JS (Day 3)
## 30 Day JS

## Description
A single page application that sets the CSS of the input value using vanilla Javascript with eventListener, dataset with suffix, name and value. The CSS utilizes the root variables (--) to use as a global variable in css, and to utilize to change the value in js

## Features
- **Range/Color Input:** Uses input element with the attribute of dataset, name, and value that is referenced by the DOM then is changed to the current value as triggered by the eventlistener. This in turn changes the padding size, the blur, or the color.

## Tech Stack
- **Core:** HTML5, CSS3, ES6+, JavaScript
- **APIs:** DOM (Document Object Model)
- **Concepts:** CSS variables, eventListener, document to reference the root, setProperty to change the property's value.

## Key Concepts Learned

- **Datasets:** Using datasets to make our own attribute that lets you use the value inside to append to the value of the other attributes.
- **CSS variables:** Learned how to create root variables that can be later manipulated with JS

## Usage

1. Clone the repository.
2. Open `css-var-js.html` in your preferred browser.
